# Projet fourni — Examen final EDW2

Ce projet PHP minimal est fourni pour l’examen final du cours EDW2.

## Objectif

L’objectif n’est pas de créer une nouvelle application, mais de démontrer les compétences suivantes :

- récupérer un projet existant avec Git ;
- modifier une information simple dans le projet ;
- créer un dépôt GitHub public personnel ;
- ajouter un nouveau remote Git ;
- envoyer le projet vers ce dépôt ;
- récupérer le projet sur N0C ;
- mettre le projet en ligne ;
- documenter les étapes dans un fichier Markdown.

## Modification demandée

Dans le fichier `index.php`, remplacez la valeur suivante :

```php
$nomEtudiant = "À remplacer par votre nom";
```
